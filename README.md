# 📊 Batch Data Processing Pipeline using Python & SQL

## 📌 Overview
Built a batch ETL pipeline to process and analyze large structured datasets using Python and MySQL. The system extracts raw data from CSV files, performs data cleaning and transformation, and loads the processed data into a relational database for analysis.

---

## 🎯 Key Features
- Batch data processing of 50K+ records  
- Data cleaning and transformation using Pandas  
- Automated ETL pipeline execution  
- Data storage using MySQL  
- Optimized queries for faster data retrieval  

---

## ⚙️ Tech Stack
- Programming: Python  
- Data Processing: Pandas  
- Database: MySQL  
- Automation: Cron / Task Scheduler  

---

## 🔄 Pipeline Workflow
1. Extract data from CSV file  
2. Clean and transform data (remove nulls, duplicates, feature creation)  
3. Load processed data into MySQL  
4. Run SQL queries for analysis  

---

## 📊 Dataset

-https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
---

## 🚀 Key Outcomes
- Processed large datasets efficiently using batch processing  
- Improved data quality by ~35% through cleaning and transformation  
- Achieved faster query performance using optimized SQL  
- Reduced manual effort through pipeline automation
